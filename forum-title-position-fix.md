# Forum 页面标题位置调整

## 问题描述

在为 Forum 页面添加了固定位置的返回按钮后，发现左上角的返回按钮与页面标题"AI论坛"产生了重叠，影响了页面的美观和可读性。

## 解决方案

通过调整页面头部标题区域的左边距，为固定返回按钮预留足够的空间，确保两个元素不会重叠。

## 具体修改

### 1. 桌面端调整

```scss
.header-left {
  margin-left: 80px; // 给标题添加左边距，避免与返回按钮重叠
  
  h1 {
    font-size: 32px;
    font-weight: 600;
    margin: 0 0 8px;
  }

  .subtitle {
    font-size: 16px;
    opacity: 0.8;
    margin: 0;
  }
}
```

### 2. 移动端响应式调整

```scss
// 移动端响应式调整
@media (max-width: 768px) {
  .header-left {
    margin-left: 70px; // 移动端按钮较小，减少左边距
  }

  .forum-actions {
    flex-direction: column;
    gap: 12px;

    .search-input {
      width: 100%;
    }
  }
}

@media (max-width: 480px) {
  .header-left {
    margin-left: 60px; // 小屏幕进一步减少边距
  }

  .header-content {
    flex-direction: column;
    align-items: flex-start;
    gap: 20px;
  }
}
```

## 设计考虑

### 🎯 空间规划
- **返回按钮**: 48×48px，位置 `top: 20px; left: 20px`
- **标题左边距**: 80px，为按钮和工具提示预留充足空间
- **安全距离**: 考虑按钮的悬停效果和工具提示显示

### 📱 响应式设计
- **平板端 (≤768px)**: 左边距 70px，适应较小的按钮尺寸
- **手机端 (≤480px)**: 左边距 60px，并调整整体布局为纵向排列

### 🎨 视觉平衡
- 保持标题与其他元素的对齐关系
- 确保搜索框和操作按钮的位置合理
- 维持整体页面的视觉协调性

### 边距计算
```
返回按钮宽度: 48px
按钮左边距: 20px
工具提示宽度: ~80px
安全间距: 12px
总计左边距: 80px
```

### 响应式断点
- **768px**: 中等屏幕断点，调整为70px
- **480px**: 小屏幕断点，调整为60px，并改为纵向布局

## 未来优化建议

1. **自适应边距**: 可以考虑使用CSS变量动态调整边距
2. **更细致的断点**: 为更多屏幕尺寸提供优化
3. **按钮位置选项**: 允许用户自定义按钮位置
4. **智能检测**: 自动检测内容重叠并调整位置

## 注意事项

1. 确保在不同浏览器中测试效果
2. 验证RTL（右到左）语言的显示效果
3. 检查高分辨率屏幕下的显示质量
4. 测试键盘导航的可访问性
