## 技术实现

### 前端技术栈
- **Vue.js 2.6.12**: 前端框架
- **Element UI 2.15.14**: UI组件库
- **Vue Router 3.4.9**: 路由管理
- **Axios 0.28.1**: HTTP客户端
- **SCSS**: 样式预处理器

## 数据模型

### 帖子数据结构
```javascript
{
  postId: 1,                    // 帖子ID
  title: "帖子标题",             // 标题
  content: "帖子内容",           // 内容
  categoryId: 1,                // 分类ID
  userId: 1,                    // 作者ID
  userName: "用户名",            // 用户名
  nickName: "昵称",              // 昵称
  avatar: "头像URL",             // 头像
  viewCount: 100,               // 浏览次数
  replyCount: 10,               // 评论数
  likeCount: 5,                 // 点赞数
  createTime: "2024-03-20",     // 创建时间
  status: "0"                   // 状态
}
```

### 评论数据结构
```javascript
{
  replyId: 1,                   // 评论ID
  postId: 1,                    // 帖子ID
  content: "评论内容",           // 内容
  parentId: 0,                  // 父评论ID
  userId: 1,                    // 用户ID
  userName: "用户名",            // 用户名
  nickName: "昵称",              // 昵称
  avatar: "头像URL",             // 头像
  createTime: "2024-03-20",     // 创建时间
  children: []                  // 子评论列表
}
```

## 使用说明

### 启动项目
```bash
cd ai-platform-web
npm install
npm run dev
```

### 访问论坛
- 论坛首页: `http://localhost:8080/forum`
- 功能测试: `http://localhost:8080/forum/test`

### 主要操作
1. **发布帖子**: 点击"发布帖子"按钮
2. **查看帖子**: 点击帖子标题
3. **发表评论**: 在帖子详情页输入评论
4. **点赞**: 点击点赞按钮
5. **搜索**: 使用顶部搜索框
6. **筛选**: 点击分类标签
