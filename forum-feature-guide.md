# AI论坛功能使用指南

## 功能概述

AI论坛是一个基于Vue.js和Spring Boot的多用户论坛系统，支持用户发布帖子、评论、点赞等社交功能。

## 主要功能

### 1. 帖子管理
- **发布帖子**: 用户可以在论坛首页点击"发布帖子"按钮创建新帖子
- **查看帖子**: 支持按分类浏览帖子，包括经验分享、问题求助、技术讨论、AI资讯
- **搜索帖子**: 支持按标题和内容搜索帖子
- **帖子详情**: 点击帖子标题可查看详细内容和评论

### 2. 评论系统
- **发表评论**: 在帖子详情页面可以发表评论
- **回复评论**: 支持对评论进行回复（嵌套评论）
- **评论管理**: 支持删除自己的评论

### 3. 互动功能
- **点赞功能**: 支持对帖子和评论进行点赞/取消点赞
- **浏览统计**: 自动统计帖子浏览次数
- **用户统计**: 显示论坛统计信息（帖子数、用户数、评论数）

### 4. 用户界面
- **响应式设计**: 支持桌面端和移动端访问
- **分类导航**: 支持按分类筛选帖子
- **侧边栏**: 显示热门标签、活跃用户等信息

## 技术架构

### 前端技术栈
- **Vue.js 2.x**: 前端框架
- **Element UI**: UI组件库
- **Vue Router**: 路由管理
- **Axios**: HTTP客户端

### 后端技术栈
- **Spring Boot**: 后端框架
- **MyBatis**: ORM框架
- **MySQL**: 数据库（当前使用模拟数据）

## 使用说明

### 1. 访问论坛
- 访问路径: `/forum`
- 或从首页导航进入论坛

### 2. 发布帖子
1. 在论坛首页点击"发布帖子"按钮
2. 填写帖子标题和内容
3. 选择帖子分类
4. 点击"发布"按钮

### 3. 查看和评论
1. 点击帖子标题进入详情页
2. 在评论区输入评论内容
3. 点击"发表评论"按钮

### 4. 点赞功能
- 在帖子详情页点击"点赞"按钮
- 支持对帖子和评论进行点赞

### 5. 搜索和筛选
- 使用顶部搜索框搜索帖子
- 点击分类标签筛选帖子

## 数据模型

### 帖子(ForumPost)
```javascript
{
  postId: 1,           // 帖子ID
  title: "帖子标题",    // 标题
  content: "帖子内容",  // 内容
  categoryId: 1,       // 分类ID
  userId: 1,           // 作者ID
  userName: "用户名",   // 用户名
  nickName: "昵称",     // 昵称
  avatar: "头像URL",    // 头像
  viewCount: 100,      // 浏览次数
  replyCount: 10,      // 评论数
  likeCount: 5,        // 点赞数
  createTime: "2024-03-20", // 创建时间
  status: "0"          // 状态
}
```

### 评论(ForumReply)
```javascript
{
  replyId: 1,          // 评论ID
  postId: 1,           // 帖子ID
  content: "评论内容",  // 内容
  parentId: 0,         // 父评论ID（0表示顶级评论）
  userId: 1,           // 用户ID
  userName: "用户名",   // 用户名
  nickName: "昵称",     // 昵称
  avatar: "头像URL",    // 头像
  createTime: "2024-03-20", // 创建时间
  children: []         // 子评论列表
}
```
